Для каждого указано 3 пункта:
- теоретический вопрос
- задание из отчёта (по номеру пункта в методичке cfd_course_26.pdf)
- Номер практического задание. Постановку практических заданий смотри по ссылке https://disk.yandex.ru/i/PP7X95akm-RcGA.

В практическом задании нужно собрать систему линейных уравнений для указанного дифура указанным методом (МКР, МКО, МКЭ).
Если вдруг в постановке что-то некорректно или каких-то данных не хватает, дополните и исправьте сами как хотите.
Для некоторых двумерных задач указаны строки СЛАУ, которые нужно собрать (всю матрицу для таких заданий не надо)
На рисунке указан масштаб. Красным -- индексы узлов. Строки матрицы должны соответствовать узлам. Нумерация с нуля.
